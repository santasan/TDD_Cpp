# TDD_Cpp
TDD By Example rewritten by Cpp

## How to execute test

### Preparation
From each folder (ex.Money/chap1)

```
mkdir build && cd build
cp ../chap1/Dollar.cpp ../
cp ../chap1/Dollar.hpp ../
cp ../chap1/MoneyTest.cpp ../
cmake ../
```

### Execute test
In build folder

```
make
./MoneyTest
```
